Δήλωση Προσβασιμότητας

Η δέσμευσή μας για ένα προσβάσιμο διαδίκτυο

Η Δέσμευσή μας

Η SafeGuard δεσμεύεται να διασφαλίζει την ψηφιακή προσβασιμότητα για άτομα με αναπηρίες. Εργαζόμαστε συνεχώς για τη βελτίωση της εμπειρίας χρήστη για όλους τους επισκέπτες του ιστοτόπου μας και εφαρμόζουμε τα σχετικά πρότυπα προσβασιμότητας.

Στόχος μας είναι η συμμόρφωση με το επίπεδο WCAG 2.2 AA (Web Content Accessibility Guidelines), ένα διεθνώς αναγνωρισμένο πρότυπο που καθορίζει πώς το διαδικτυακό περιεχόμενο μπορεί να γίνει πιο προσβάσιμο σε άτομα με αναπηρίες.

Μέτρα Προσβασιμότητας

Έχουμε λάβει τα ακόλουθα μέτρα για να διασφαλίσουμε την προσβασιμότητα του ιστοτόπου μας:

  • Πλοήγηση με πληκτρολόγιο: Όλες οι λειτουργίες του ιστοτόπου είναι προσβάσιμες μέσω πληκτρολογίου. Μπορείτε να πλοηγηθείτε χρησιμοποιώντας το Tab, τα βέλη και το Enter.
  • Σύνδεσμος παράκαμψης: Παρέχουμε σύνδεσμο "Μετάβαση στο περιεχόμενο" στην αρχή κάθε σελίδας για γρήγορη πρόσβαση στο κύριο περιεχόμενο.
  • Ορατή εστίαση: Όλα τα διαδραστικά στοιχεία (σύνδεσμοι, κουμπιά, πεδία φορμών) έχουν ευδιάκριτο περίγραμμα εστίασης όταν επιλέγονται.
  • Αντίθεση χρωμάτων: Χρησιμοποιούμε χρώματα με επαρκή αντίθεση (τουλάχιστον 4.5:1 για κανονικό κείμενο) ώστε το περιεχόμενο να είναι ευανάγνωστο.
  • Σημασιολογική HTML: Χρησιμοποιούμε σημασιολογικά στοιχεία HTML (header, nav, main, footer, article, section) για καλύτερη κατανόηση της δομής από βοηθητικές τεχνολογίες.
  • Ιεραρχία επικεφαλίδων: Κάθε σελίδα έχει μία επικεφαλίδα H1 και λογική ιεραρχία επικεφαλίδων (H2, H3) για εύκολη πλοήγηση.
  • Εναλλακτικό κείμενο εικόνων: Όλες οι ουσιαστικές εικόνες έχουν περιγραφικό εναλλακτικό κείμενο (alt text). Οι διακοσμητικές εικόνες επισημαίνονται κατάλληλα.
  • Προσβάσιμες φόρμες: Τα πεδία των φορμών έχουν σαφείς ετικέτες, οδηγίες και μηνύματα σφάλματος που ανακοινώνονται στους αναγνώστες οθόνης.
  • Ανταπόκριση σε κινητές συσκευές: Ο ιστότοπος είναι πλήρως responsive και λειτουργεί σε όλα τα μεγέθη οθόνης.
  • Σεβασμός προτιμήσεων κίνησης: Σεβόμαστε τη ρύθμιση "prefers-reduced-motion" για χρήστες που προτιμούν μειωμένη κίνηση.
  • Ετικέτες ARIA: Χρησιμοποιούμε ετικέτες ARIA όπου είναι απαραίτητο για να βελτιώσουμε την προσβασιμότητα δυναμικού περιεχομένου.

Γνωστοί Περιορισμοί

Παρά τις προσπάθειές μας, ενδέχεται να υπάρχουν κάποιοι περιορισμοί:

  • Περιεχόμενο τρίτων: Η φόρμα επικοινωνίας χρησιμοποιεί την υπηρεσία Formspree, την οποία δεν ελέγχουμε πλήρως. Ωστόσο, έχουμε σχεδιάσει τη φόρμα με γνώμονα την προσβασιμότητα.
  • Παλαιότεροι browsers: Ορισμένες δυνατότητες CSS ενδέχεται να μην υποστηρίζονται πλήρως σε πολύ παλιές εκδόσεις browsers.

Εργαζόμαστε συνεχώς για τη βελτίωση της προσβασιμότητας και την αντιμετώπιση τυχόν προβλημάτων.

Τεχνικές Προδιαγραφές

Η προσβασιμότητα του ιστοτόπου βασίζεται στις ακόλουθες τεχνολογίες:

  • HTML5
  • CSS3
  • JavaScript (ES6+)
  • Bootstrap 5
  • WAI-ARIA 1.2

Αυτές οι τεχνολογίες χρησιμοποιούνται σε συνδυασμό με τις δυνατότητες προσβασιμότητας των σύγχρονων browsers και βοηθητικών τεχνολογιών.
