Πολιτική Απορρήτου

Πώς διαχειριζόμαστε τα δεδομένα σας

Εισαγωγή

Η SecureGuard σέβεται το απόρρητό σας και δεσμεύεται να προστατεύει τα προσωπικά σας δεδομένα. Αυτή η πολιτική απορρήτου εξηγεί πώς συλλέγουμε, χρησιμοποιούμε και προστατεύουμε τις πληροφορίες που μας παρέχετε μέσω του ιστοτόπου μας.

Τελευταία ενημέρωση: Ιανουάριος 2026

Δεδομένα που Συλλέγουμε

Δεδομένα από τη Φόρμα Επικοινωνίας

Όταν χρησιμοποιείτε τη φόρμα επικοινωνίας του ιστοτόπου μας, συλλέγουμε τα εξής δεδομένα:

  • Ονοματεπώνυμο: Για να σας απευθυνθούμε στην απάντησή μας
  • Διεύθυνση email: Για να σας απαντήσουμε
  • Τηλέφωνο (προαιρετικό): Αν επιθυμείτε τηλεφωνική επικοινωνία
  • Μήνυμα: Το περιεχόμενο του αιτήματός σας

Τεχνικά Δεδομένα

Ο ιστότοπος μας δεν χρησιμοποιεί cookies ούτε εργαλεία ανάλυσης (analytics) που παρακολουθούν την πλοήγησή σας. Φιλοξενείται στο GitHub Pages, το οποίο μπορεί να καταγράφει βασικές τεχνικές πληροφορίες (όπως διεύθυνση IP) για λόγους ασφαλείας και λειτουργίας.

Επεξεργασία Φόρμας Επικοινωνίας

Η φόρμα επικοινωνίας του ιστοτόπου μας χρησιμοποιεί την υπηρεσία Formspree για την αποστολή των μηνυμάτων σας.

Τι σημαίνει αυτό;

  • Όταν υποβάλλετε τη φόρμα, τα δεδομένα σας αποστέλλονται στο Formspree
  • Το Formspree τα προωθεί στο email μας
  • Το Formspree έχει δική του πολιτική απορρήτου: formspree.io/legal/privacy-policy

Χρήση Δεδομένων

Χρησιμοποιούμε τα δεδομένα που μας παρέχετε αποκλειστικά για:

  • Να απαντήσουμε στο αίτημα ή ερώτημά σας
  • Να σας παρέχουμε πληροφορίες ή προσφορές που ζητήσατε
  • Να σας εξυπηρετήσουμε αν γίνετε πελάτης μας

Δεν πουλάμε, ενοικιάζουμε ή μοιραζόμαστε τα προσωπικά σας δεδομένα με τρίτους για σκοπούς μάρκετινγκ.

Διατήρηση Δεδομένων

Διατηρούμε τα δεδομένα επικοινωνίας για όσο χρόνο είναι απαραίτητο για να ολοκληρώσουμε το αίτημά σας και να τηρήσουμε τις νομικές μας υποχρεώσεις. Συνήθως, τα δεδομένα διατηρούνται για διάστημα έως 2 ετών, εκτός αν υπάρχει συνεχής εμπορική σχέση.

Τα Δικαιώματά σας

Σύμφωνα με τον Γενικό Κανονισμό Προστασίας Δεδομένων (GDPR), έχετε τα εξής δικαιώματα:

  • Πρόσβαση: Να ζητήσετε αντίγραφο των δεδομένων που διατηρούμε για εσάς
  • Διόρθωση: Να ζητήσετε διόρθωση ανακριβών δεδομένων
  • Διαγραφή: Να ζητήσετε διαγραφή των δεδομένων σας
  • Περιορισμός: Να ζητήσετε περιορισμό της επεξεργασίας
  • Φορητότητα: Να λάβετε τα δεδομένα σας σε κοινό μορφότυπο
  • Εναντίωση: Να εναντιωθείτε στην επεξεργασία των δεδομένων σας

Για να ασκήσετε οποιοδήποτε από αυτά τα δικαιώματα, επικοινωνήστε μαζί μας στο privacy@alarm-systems.gr.

Ασφάλεια Δεδομένων

Λαμβάνουμε κατάλληλα τεχνικά και οργανωτικά μέτρα για την προστασία των δεδομένων σας, συμπεριλαμβανομένων:

  • Κρυπτογραφημένη σύνδεση (HTTPS) για την υποβολή φορμών
  • Περιορισμένη πρόσβαση στα δεδομένα μόνο σε εξουσιοδοτημένο προσωπικό
  • Τακτικός έλεγχος των διαδικασιών ασφαλείας

Cookies

Ο ιστότοπος μας δεν χρησιμοποιεί cookies για παρακολούθηση ή ανάλυση. Ενδέχεται να χρησιμοποιούνται τεχνικά απαραίτητα cookies από την υπηρεσία φιλοξενίας (GitHub Pages) για τη βασική λειτουργία του ιστοτόπου.

Εξωτερικοί Σύνδεσμοι

Ο ιστότοπός μας μπορεί να περιέχει συνδέσμους προς εξωτερικούς ιστότοπους. Δεν είμαστε υπεύθυνοι για τις πρακτικές απορρήτου αυτών των ιστοτόπων. Σας συνιστούμε να διαβάζετε τις πολιτικές απορρήτου κάθε ιστοτόπου που επισκέπτεστε.

Αλλαγές στην Πολιτική

Ενδέχεται να ενημερώσουμε αυτή την πολιτική απορρήτου κατά καιρούς. Οποιεσδήποτε αλλαγές θα δημοσιεύονται σε αυτή τη σελίδα με ενημερωμένη ημερομηνία. Σας συνιστούμε να ελέγχετε περιοδικά αυτή τη σελίδα.
